# Modelo de Score de Propostas

Este projeto tem como objetivo construir um modelo de Machine Learning para prever a probabilidade de contratacao de propostas.

## Objetivo

Criar um modelo que classifique propostas com maior chance de conversao, auxiliando na priorizacao comercial.

## Tecnologias usadas

- Python
- Pandas
- Scikit-learn
- Jupyter Notebook
- Matplotlib
- Git/GitHub

## Estrutura do projeto

- `notebooks/`: analises exploratorias e testes
- `src/`: scripts Python do projeto
- `reports/`: metricas e resultados
- `data/`: bases locais nao versionadas
